## üìù Creation Steps

### Step 1: Create Shops
1. Login as admin or register as seller
2. Go to shop registration page
3. Fill in all shop details from above
4. Submit and verify email if required

### Step 2: Create Products for Each Shop
1. Login to respective shop account
2. Navigate to "Add Product" section
3. Fill in all product details
4. Upload thumbnail image (use provided Unsplash URLs or download and upload)
5. Add multiple images if possible
6. Set categories from the dropdown
7. Add colors, sizes, and tags as specified
8. Save and activate product

### Step 3: Verify
- Check shop profiles are visible
- Ensure all products are listed
- Test product search and filtering
- Verify product detail pages
- Test add to cart functionality
